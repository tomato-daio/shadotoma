# シャドとま (Shadotoma)

英語シャドーイング練習用PWA（個人開発）。VOA Learning Englishのパブリックドメイン教材でリスニング→オーバーラッピング→シャドーイング→録音提出という練習フローを回し、録音をブラウザ内蔵のWhisper（音声認識AI）で自動採点します。

作者: tomato-daio

## 特徴

- **個人利用・ランニングコスト0円・iPhone完結**を前提に設計。サーバーは持たず、GitHub Pagesで配信される静的サイトのみで完結します。
- **添削もすべて端末内で完結**。録音した音声はサーバーへ送らず、ブラウザ内で動くWhisper（`@huggingface/transformers` + ONNX Runtime Web、WASM実行）が文字起こしし、スクリプトとの一致率を計算します。
- オフラインでも一度読み込んだ教材は聴けます（Service Workerによるキャッシュ）。

## 機能一覧

- **今日の練習**: 進行中教材と今日のステップ、練習開始、連続日数バッジ
- **練習フローウィザード**: 1日目はリスニング→スクリプト確認→オーバーラッピング、2〜4日目はシャドーイング→録音提出、という順にガイド（自由にスキップ・前後移動も可能）
- **プレーヤー**: 再生速度変更（0.5〜2.0倍・ピッチ維持）、ABリピート、3秒巻き戻し、ループ回数カウント、スクリプト表示切替
- **録音・聴き比べ**: マイク録音、レベルメーター表示、自分の録音とお手本の聴き比べ、録り直し
- **自動添削**: 録音をオンデバイスのWhisperで文字起こしし、スクリプトと単語単位で整列（一致/欠落/置換を色分け）。一致率・WPM（発話速度）・Good Point/Development Pointを自動生成
- **教材ライブラリ**: VOA Learning Englishのバンドル教材（レベル・カテゴリでフィルタ）に加え、手持ちの音声ファイルをローカル取り込み可能
- **進捗管理**: 連続学習日数、提出履歴、スコア推移
- **データのエクスポート/インポート**: 端末内データをJSONファイルとしてバックアップ・復元

## 開発コマンド

```bash
npm install       # 依存インストール（初回のみ）
npm run dev       # 開発サーバー起動（http://localhost:5173）
npm run build     # 型チェック + 本番ビルド（dist/に出力）
npm run preview   # ビルド済みdist/をローカルで確認（http://localhost:4173）
npm test          # Vitestでテスト実行
npm run fetch-voa -- --level 1 --count 5   # VOA教材を取得してpublic/materials/に追加
```

## 教材の追加手順

### VOA教材（バンドル教材として追加）

```bash
npm run fetch-voa -- --level <1|2|3> --count <件数>
```

`scripts/fetch-voa.mjs` がVOA Learning English（米国政府制作＝パブリックドメイン）から記事の音声とトランスクリプトを取得し、`public/materials/index.json` と `public/materials/audio/*.mp3` に追記します（既存分は上書きせず、重複URLはスキップ）。取得後は通常どおりコミットしてデプロイすれば全端末で利用できます。

### 手持ちの音声（ローカル取り込み）

アプリの「教材」タブから音声ファイル（TED音源など著作権のあるものでも可）を選んで取り込めます。この場合、音声とスクリプトはその端末のIndexedDB内にのみ保存され、リポジトリには一切含まれません（他の端末とは共有されません）。

## iPhoneでの利用手順

1. iPhoneのSafariでGitHub PagesのURL（`https://tomato-daio.github.io/shadotoma/`）を開く
2. 共有ボタン（□に↑）をタップ →「ホーム画面に追加」を選択
3. ホーム画面に追加されたアイコンからアプリとして起動
4. 録音ステップで初回のみマイク使用の許可ダイアログが出るので「許可」を選択
5. 初回の添削実行時、Whisperモデル（数十MB）のダウンロードが走ります。一度ダウンロードすればブラウザにキャッシュされ、以降はオフラインでも添削できます

イヤホンを付けてお手本を流しながら録音すると、お手本の音声が録音に混入せずきれいに録れます。

## データの保存先について

このアプリは録音・進捗・教材データを含め、すべての情報を端末内のIndexedDBにのみ保存します。サーバーは存在せず、外部へデータを送信する処理も一切ありません。添削（音声認識）もブラウザ内蔵のAIモデルで完結し、録音データが外部に送られることはありません。

複数端末間でデータを引き継ぎたい場合は、「設定」タブのエクスポート機能でJSONファイルを書き出し、別端末でインポートしてください。

## 技術スタック

Vite 7 + React 18 + TypeScript (strict) + Tailwind CSS v4 + vite-plugin-pwa + IndexedDB (idb) + Zustand + `@huggingface/transformers`（Whisper tiny.en, WASM実行）。詳細な設計は [DESIGN.md](./DESIGN.md) を参照してください。
