# Azure発音評価のセットアップ手順

シャドとまには、Microsoft Azureの「音声サービス（Speech）」を使った発音スコア機能があります。
**完全に任意の機能**です。設定しなくても、これまでどおりの添削（Whisperによる一致率・WPM判定）は
そのまま使えます。設定すると、判定結果画面に「発音スコア（総合・正確さ・流暢さ・韻律・完全性）」
が追加で表示されるようになります。

このページの手順は、Azureの画面構成が変わると多少ズレる場合があります。表示名が多少違っても、
おおむね同じ手順で進められます。

## 1. Azureアカウントを作る（無料）

1. ブラウザで https://azure.microsoft.com/ja-jp/free/ を開く。
2. 「無料で始める」または「今すぐ試す」ボタンをクリックする。
3. 案内に従ってサインアップする（お持ちのMicrosoftアカウント/Outlook宛メールアドレスでも新規作成でも可）。
   - 本人確認のためクレジットカード情報の入力を求められることがあります。これはAzure全体の
     アカウント確認のためで、この後の手順で選ぶ「Free（無料）」プランを使う限り課金は発生しません。
4. サインアップが完了したら、次のステップへ進む。

## 2. 音声サービス（Speech）リソースを作成する

1. ブラウザで https://portal.azure.com/ を開く（Azure Portal）。
2. 画面左上のメニューから「リソースの作成」をクリックする。
3. 検索欄に「音声サービス」または「Speech」と入力し、検索結果から「音声サービス」
   （提供元: Microsoft）を選ぶ。
4. 「作成」ボタンをクリックする。
5. 作成フォームで、以下の項目を入力・選択する。
   - **サブスクリプション**: 自分のサブスクリプション（無料試用など）を選ぶ。
   - **リソースグループ**: 「新規作成」をクリックし、任意の名前（例: `shadotoma-rg`）を付けて作成する。
   - **リージョン**: **「Japan East（東日本）」を選ぶ**（推奨。日本国内から使う場合、応答が速い）。
   - **名前**: 任意の名前（例: `shadotoma-speech`）を入力する。
   - **価格レベル**: **「Free F0」を選ぶ**（重要。下の「無料枠について」を参照）。
     `Standard S0` など他のプランは有料なので選ばないこと。
6. 内容を確認し、「確認と作成」→「作成」の順にクリックする。
7. デプロイが完了するまで待つ（数十秒〜数分程度）。完了したら「リソースに移動」をクリックする。

## 3. APIキーとリージョンを確認する

1. 作成したリソースのページで、左側メニューの「リソース管理」→「キーとエンドポイント」を開く。
2. 「キー 1」の右側にある文字列（長い英数字）をコピーする。これがAPIキー。
3. 同じ画面の「場所/地域」欄の値（例: `japaneast`）も確認しておく（手順2-5で選んだリージョンと
   同じになっているはず）。

**注意**: このキーは他人に見せたり、SNS・チャット・ファイル共有などで送ったりしないこと。
シャドとま側でも、バックアップのエクスポートファイルにはこのキーを含めない設計になっている
（キーは端末内のブラウザにのみ保存される）。

## 4. シャドとまに設定する

1. シャドとまを開き、下タブの「設定」をタップする。
2. 「発音スコア（Azure・任意）」セクションまでスクロールする。
3. 「APIキー」欄に、手順3-2でコピーしたキーを貼り付ける。
4. 「リージョン」欄で、手順3-3で確認したリージョン（例: 東日本 / japaneast）を選ぶ。
5. 「接続テスト」をタップする。「接続に成功しました。」と表示されればキーは有効。
   失敗する場合は、キーの貼り付け間違い・リージョンの選び間違いがないか確認する。
6. 「保存」をタップする。

以上で設定完了。次回の提出（録音の添削）から、Whisperの採点に続けて発音スコアが自動で
計算されるようになる。

## 無料枠について

- 「Free F0」プランは、**1か月あたり音声認識5時間分まで無料**（2026年時点。最新の条件は
  Azureの価格ページで必ず確認すること）。
- シャドとまの1回の提出は数十秒程度なので、毎日数分の練習であれば月5時間には収まらず、
  実質0円で使い続けられる。
- Free F0を選んでいる限り、無料枠を超えても自動的に高額課金される心配はない
  （その月はそれ以上の音声認識ができなくなるだけ）。

## 使うのをやめたい場合

シャドとまの設定画面の「発音スコア（Azure・任意）」セクションにある「削除」ボタンを押すと、
保存したAPIキーが端末から削除される。以降は通常のWhisper採点のみに戻る
（Azureのリソース自体を削除したい場合は、Azure Portal側で該当リソースを削除する）。

## データの扱いについて

- 発音スコアの採点を実行したときだけ、**採点対象の提出音声とスクリプト文章**がAzureへ送信される。
- それ以外の学習データ（他の録音・進捗・教材など）は、これまでどおり端末内(IndexedDB)にのみ
  保存され、外部へ送信されることはない。
- APIキーは端末内のブラウザにのみ保存され、シャドとまの開発者を含め、どこにも送信されない。
